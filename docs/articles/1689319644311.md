> 环境需求：**Node.js v16** 以上版本

### 拉取源码

```
git clone https://github.com/palxiao/poster-design.git
cd poster-design
```

### 安装依赖

```
npm run prepared
```

#### 本地运行

```
npm run serve
```

> 将会同时运行前端界面与图片生成服务：
> 
> ![](../images/2023-7-16-1689498291322.png)

#### 运行结果

![](../images/2023-7-16-1689500112694.gif)

#### 打包前端页面

```
npm run v-build
```

#### 打包图片生成服务

```
cd sreenshot
npm run build
```
