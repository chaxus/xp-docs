## 编辑器常用操作

### 快捷键

- **保存**：Ctrl / Command + **S**
- **复制（选中元素）**：Ctrl / Command + **C**
- **粘贴**：Ctrl / Command + **V**
- **多选**：按住 Shift 或 Ctrl / Command 然后**鼠标点选**
- **组合成组（多选时）**：Ctrl / Command + **G**

### 参考线

- 点击状态栏中的“标尺”按钮创建画布上的**标尺**

![](../images/2023-7-17-1689560709852.png)

- 从标尺中**拖出**辅助线，拖回标尺中**删除**辅助线

![](../images/2023-7-17-1689561014327.gif)

## 图片

### 缩放和裁剪

- 点击图片，选中四角拖拽缩放大小
- 选择右侧“裁剪”
